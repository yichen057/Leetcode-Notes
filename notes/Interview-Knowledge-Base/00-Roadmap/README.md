# Roadmap 学习路线

> Target: SDE Intern / New Grad offers. 按优先级安排背诵顺序，配合间隔重复。

## 🗺️ Priority Order 优先级

| Phase | Topics | Why 为什么 |
|-------|--------|-----------|
| 1️⃣ 每天都碰 | 03-12 算法 + 99-CheatSheets | 算法是 intern/NG 面试的大头，模板要烂熟 |
| 2️⃣ 面试前 2 周 | 01-Python / 02-Java（选主语言）| "Tell me about GIL/JVM" 是热身高频题 |
| 3️⃣ 面试前 2 周 | 17-Behavioral | 每家都考，STAR 故事提前写好背熟 |
| 4️⃣ 面试前 1 周 | 14-OS + 15-Network | 八股经典，背就完事 |
| 5️⃣ 按需 | 16-SystemDesign | intern 少考，NG/E4+ 必考 |
| 6️⃣ 面试前 3 天 | 19/20/21 公司专题 | 面哪家看哪家：Amazon 必看 LP |

## 📅 Weekly Plan 周计划模板

- **Sat**: mock interview（英文出声回答）
- **Sun**: 复盘本周错题 + 更新 [进度表](#progress)
