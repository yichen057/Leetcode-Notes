## ✍️ Note Template 笔记模板

所有笔记遵循统一模板，见 [TEMPLATE.md](./TEMPLATE.md)。核心格式：

```markdown
### Q: <English question>

**A:** <English answer — what you'd say in the interview>

**中文**: <中文讲解，帮助理解和记忆>

**Follow-up**: <常见追问>
```

## 🔁 How to Memorize 背诵方法

1. **首遍理解**：读中文讲解，确保理解原理
2. **遮答自测**：只看 Q，用英文口述 A，再对照
3. **间隔重复**：1 天 → 3 天 → 7 天 → 14 天复习
4. **模拟面试**：对着镜子/录音用英文完整回答，控制在 1-2 分钟
